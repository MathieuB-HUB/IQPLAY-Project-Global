# IQPLAY-Project-Global

## À propos du projet

IQPlayLib est une bibliothèque Java conçue pour simplifier la création de jeux vidéo interactifs. Elle fournit des outils prêts à l'emploi pour gérer des sessions de jeu, des niveaux, des scores et des questions, permettant ainsi aux développeurs de se concentrer sur la logique et le design de leurs jeux. 

Que vous souhaitiez créer un quiz, un jeu éducatif ou un jeu basé sur des défis, IQPlayLib offre une base modulaire pour démarrer rapidement.

## Architecture du projet et Repositories

Le projet est segmenté en plusieurs dépôts pour séparer la logique de la bibliothèque, le serveur et les interfaces de jeux :

### Cœur du projet
* **[Library Java](https://github.com/IQPlay/library)** : Le framework principal contenant la logique de gestion des jeux.
* **[Backend](https://github.com/IQPlay/backend)** : Serveur dédié aux jeux de test et à la persistance des données.

### Applications et Jeux de test
* **[Frontend Calcul Mental](https://github.com/IQPlay/calculMental-frontend)** : Interface utilisateur pour un jeu de calcul mental basé sur la bibliothèque.
* **[Frontend WikiGame](https://github.com/IQPlay/wikigame-frontend)** : Interface utilisateur pour le jeu WikiGame.
